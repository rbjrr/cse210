Eternal Quest - Simple Console Project
Files:
- Goal.cs
- SimpleGoal.cs
- EternalGoal.cs
- ChecklistGoal.cs
- GoalManager.cs
- Program.cs

How to run:
1. Create a new C# console project or place files into a folder and compile:
   csc *.cs
   or open in Visual Studio and run.
2. The program saves to "eternalquest_save.txt" in the running directory.
3. The project follows OOP principles: Abstraction, Encapsulation, Inheritance, Polymorphism.

Extras:
- Leveling system: every 1000 points you gain a level (creativity to exceed requirements).
